3.1. Создание первого приложения
https://app.purpleschool.ru/courses/19/sections/293/lessons/2213

Репозиторий проекта
https://github.com/AlariCode/15-go-basics-1

Цель урока: Научиться создавать и запускать базовое приложение на языке программирования Go.

Шаги:

1 Создание проекта:

Создайте новую папку для проекта, используя команду mkdir go-demo в консоли или же стандартные средства вашей операционной системы (Finder на macOS, Проводник на Windows, файловый менеджер на Linux).

2 Настройка среды разработки:

Откройте папку проекта в Visual Studio Code. Это можно сделать, перетащив папку в окно редактора, или используя команду code . в терминале, предварительно перейдя в директорию проекта (cd go-demo).

3 Создание основного файла:

В корневой директории проекта создайте файл main.go.
В файле main.go укажите имя пакета как package main.
Определите функцию main внутри файла. Она будет точкой входа вашего приложения.

4 Программирование функционала:

Для вывода текста используйте пакет fmt и его функцию Println("привет") или Printf("привет") внутри функции main.

5 Запуск приложения:

Откройте терминал в Visual Studio Code или используйте отдельный терминал.

Запустите приложение командой go run main.go.

После выполнения команды в терминале отобразится сообщение "привет", что означает успешный запуск вашего первого приложения.
